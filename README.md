# SZ://DEV - Personal Portfolio

A brutalist, high-performance personal portfolio website for **Shivrajsinh Zala**, built with modern web technologies. Designed to be bold, fast, and highly interactive.

## 🚀 Live Site
[shivrajsinhzala.github.io](https://shivrajsinhzala.github.io)

## 🛠 Tech Stack
* **HTML5** - Semantic and accessible markup.
* **Tailwind CSS** - Utility-first styling with custom brutalist design tokens.
* **GSAP (GreenSock)** - Advanced scroll-triggered animations and horizontal scrolling.
* **Lenis** - Butter-smooth scrolling experience.
* **Lucide Icons** - Clean and minimal iconography.

## ✨ Features
* **Brutalist Design:** High-contrast borders, bold colors, and a raw, unpolished aesthetic.
* **Horizontal Scroll:** Custom GSAP horizontal scroll implementation for the projects section.
* **Smooth Scrolling:** Integrated Lenis smooth scroll for a premium feel.
* **Custom Cursor:** Interactive cursor that changes state on hover.
* **Easter Eggs:** Hidden `_TERMINAL` interface with working commands.
* **Fully Responsive:** Optimized for desktop, tablet, and mobile devices.
* **SEO Optimized:** Fully configured meta tags, JSON-LD schema, and Open Graph data for maximum search engine visibility.

## 📦 Development

### Prerequisites
To make changes to the styling, you will need Node.js installed to run the Tailwind CSS compiler.

### Build Tailwind CSS
The project uses a static CSS build for maximum performance. If you edit the Tailwind classes in the HTML files, recompile the CSS by running:
```bash
npx tailwindcss -i src/input.css -o assets/css/tailwind.css --minify
```
